# AdvanSpring
Spring을 독학해보면서 작성한 프로젝트입니다.
어떤 목적보다는 Spring에 있는 여러 기능을 살펴보기 위해 작성되었습니다.

## Dependency
`@RestController`, `@GetMapping` 등의 web 기능을 이용하려면 아레의 dependency가 필요함
```kt
implementation("org.springframework.boot:spring-boot-starter-web")
```
